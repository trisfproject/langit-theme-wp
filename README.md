# Langit WordPress Theme

## Theme Information

**Theme Name:** Langit

**Description:** Modern lightweight corporate WordPress theme for industrial technology, security systems, networking, and company profile websites.

## Theme Features

- Responsive Design
- Mobile & Tablet Friendly
- Gutenberg Compatible
- Lightweight Architecture
- SEO Friendly
- Modern Corporate UI
- Blog Support
- Products/Services Pages
- Sky Blue Branding

## Technology Stack

- WordPress
- PHP
- CSS
- JavaScript
